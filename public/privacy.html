Privacy Policy

Lotfy's AI Prompts for Media Buyers — Chrome Extension

Last updated: June 1, 2026

Summary

This extension collects nothing. No personal data, no analytics, no telemetry, no tracking, no account required. Your selected text is processed entirely on your device and never sent over the network.

1. What this extension does

Lotfy's AI Prompts for Media Buyers is a free Chrome extension that provides a curated library of Arabic-language AI prompts for media buying workflows. You can browse prompts in the popup or right-click selected text on any webpage to copy a prompt with your selection injected at the right place, ready to paste into Claude, ChatGPT, Gemini, or any other AI tool.

2. Information we collect

None. The extension does not collect, transmit, or store any personal information. Specifically:

  • No user accounts or sign-up
  • No analytics or usage tracking
  • No third-party trackers or advertising SDKs
  • No collection of selected text — when you right-click selected text, the injection happens locally in your browser and is written to your clipboard. The text never leaves your device.
  • No collection of browsing history
  • No collection of IP addresses, device fingerprints, or any identifiers

3. Local storage on your device

The extension uses Chrome's chrome.storage.local API to cache two pieces of data on your device only:

  • Prompts catalog: a copy of the public JSON file (prompts-for-media-buyers.lotfy.ai/v1/prompts.json) so the extension works offline after first load.
  • Last fetch timestamp: the time when the catalog was last refreshed (used to display "Last checked" in the Options page).

This data is stored locally on your device, is never transmitted anywhere, and is automatically removed when you uninstall the extension.

4. Network requests

The extension makes one type of outbound request only: a GET request to the public JSON catalog at:

https://prompts-for-media-buyers.lotfy.ai/v1/prompts.json

This file is hosted on Cloudflare and contains the same content for every user. The request happens:

  • When the extension is first installed
  • When Chrome starts up (to check for updates)
  • When you click "Refresh" in the Options page

No request body, headers identifying you, or query parameters tied to your identity are sent — only the standard browser User-Agent header that Chrome attaches to all web requests.

5. Third parties

The only third party involved in serving this extension is Cloudflare, which hosts the public catalog file. Cloudflare's privacy policy is available at cloudflare.com/privacypolicy. Cloudflare may log standard HTTP request metadata (IP, timestamp, user agent) for security and performance purposes, as any web server would. We have no access to or control over these logs.

No data is shared with any other third party. The extension does not use Google Analytics, Mixpanel, Sentry, Segment, or any other analytics or telemetry service.

6. Permissions explained

The extension requests the following Chrome permissions:

  • contextMenus: to add the "AI Prompts" item to your right-click menu.
  • storage: to cache the prompts catalog locally on your device (Section 3).
  • scripting: required by Chrome's Manifest V3 for the offscreen document used to write to the clipboard.
  • offscreen: to create a hidden document that handles clipboard writes (a Manifest V3 requirement; service workers cannot access the clipboard API directly).
  • host_permissions for https://prompts-for-media-buyers.lotfy.ai/*: to fetch the public catalog file. The extension does not request access to any other website.

7. Children's privacy

The extension is not directed at children under 13. We do not knowingly collect any information from children (in fact, we do not collect any information from anyone — see Section 2).

8. Changes to this policy

If we ever update this policy, the "Last updated" date at the top of this page will change. Any material change in data practices will also be reflected in an updated extension version. Since the extension currently collects no data, the only kind of change foreseeable is a clarification or correction.
