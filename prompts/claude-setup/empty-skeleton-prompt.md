# Role
أنت [اكتب هنا مين Claude بالنسبة للـtask ده — مثلاً: Senior Media Buyer متخصص في e-commerce على Meta Ads].

# Context
[اكتب هنا الخلفية اللي Claude محتاج يعرفها قبل ما يبدأ:
- المنتج/الخدمة وإيه اللي بتميزها
- الجمهور (سن، اهتمامات، سلوك شرائي)
- الـmarket (مصر/خليجي/إلخ)
- أي أرقام أو نتائج سابقة مهمة]

# Task
[اكتب هنا الطلب المحدد. فعل واحد، نتيجة واحدة. مثلاً:
"اكتبلي 5 نسخ primary text لإعلان Facebook للمنتج ده."]

# Constraints
[اكتب هنا القيود الحقيقية:
- اللغة (عربي مصري/فصحى/إنجليزي)
- الطول (مثلاً: كل نسخة ما تزيدش عن 90 كلمة)
- الـtone (مثلاً: مباشر بدون مبالغة)
- الممنوعات (مثلاً: ممنوع كلمات زي "الأفضل"، "الأرخص"، "حصري")
- أي compliance (مثلاً: ممنوع وعود طبية)]

# Format
[اكتب هنا شكل الـoutput المطلوب. مثلاً:
"جدول من 3 أعمدة: رقم النسخة | النص | الـangle اللي بتركز عليه (سعر/فايدة/خوف الفوات).
في الآخر، اقتراح أي نسختين أبدأ بيهم A/B test وليه."]
